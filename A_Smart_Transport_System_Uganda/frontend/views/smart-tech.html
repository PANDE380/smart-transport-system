Smart technology

Practical digital tools that make transport easier to run.

STS uses live maps, route support, wallet tracking, and chatbot assistance to make the platform easier to use without overwhelming the user experience.

Technology in the product

Each tool supports a real part of the transport workflow, from booking to support and reporting.

Route support

Pickup and destination selection connect directly to map routing so the fare and travel estimate feel grounded and clear.

Live service visibility

Operations pages surface active vehicles, coverage points, and feed activity without forcing the user to refresh constantly.

Shared wallet system

Passengers can track top ups, spending, and expected balance changes from a single wallet view.

Assistant support

The STS assistant helps answer user questions about booking, pricing, support, and onboarding.

Design approach

  • Keep the interface readable for passengers and staff.
  • Use live data only where it helps the decision in front of the user.
  • Make safety, support, and payments easy to find.
